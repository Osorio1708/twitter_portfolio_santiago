## Add file .env

You must add file .env in root folder

## NodeJs

If you dont have NodeJs, you must dowload that from this page https://nodejs.org/en/

## Add file .env

If you dont have Nest/cli, you must install that

```bash
$ npm install -g @nestjs/cli
```
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# test
$ npm run test


```
